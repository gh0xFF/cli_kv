# cli_kv

App is a simple key-value cli storage

To start using need to set environment variables where storage will be stored

## example for mac os

`
export FOLDER_PATH=/Users/&User/.kv_cli
`

`
export FILE_PATH=/Users/&User/.kv_cli/storage
`

## to start using
`
cli_kv -h
`

## to build
`
cargo build -r
`
